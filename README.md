Шаг 1: Установите Docker Desktop
1)Скачайте Docker Desktop с официального сайта: https://www.docker.com/products/docker-desktop/
2)Установите как обычную программу

Шаг 2: Скачайте файлы программы
1)Скачайте архив
2)Распакуйте скачанный архив в любую папку на компьютере

Шаг 3: Запустите программу
1)Откройте папку в любой среде разработки
2)Запустите установку одной командой в терминале:
docker-compose up -d
3)Затем перейдите по ссылке:
http://localhost:3000/

У вас автоматически развернулась база данных, бэкэнд и фронтенд
